# Projeto de Conclusão de Curso - Etec

Este é um projeto **Full Stack** desenvolvido como Trabalho de Conclusão de Curso. O sistema integra uma interface web com um servidor backend e banco de dados para gestão de informações.

## 🛠️ Tecnologias Utilizadas
* **Backend:** Python
* **Banco de Dados:** MySQL (scripts na pasta `/MySql`) e SQLite
* **Frontend:** HTML, CSS e JavaScript (na pasta `/public`)

## 📁 Estrutura do Projeto
* `app.py` / `servidor.py`: Lógica principal do servidor.
* `MySql/`: Modelagem e scripts do banco de dados.
* `public/`: Interface visual do usuário.
* `login_profissionais.py`: Módulo de autenticação.

## 🚀 Como executar
1. Configure o banco de dados MySQL usando os scripts na pasta `/MySql`.
2. Execute o servidor com o comando: `python app.py`.
